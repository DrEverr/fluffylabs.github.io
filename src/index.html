Dynamic SCALE / JAM Decoder

This handy tool decodes tricky SCALE-encoded bytes in real-time. You can tweak type definitions on the fly and use pre-loaded JAM types to auto-fill expected fields and names.

@ github

Gray Paper Decoder

A personal guide to decode the cryptic Gray Paper. It lists Greek symbols with explanations, provides screenshots with descriptions, and offers an interactive website to hover over formulas for quick insights.

@ github

PVM Disassembler

Transforms PVM code into readable instructions. Perfect for debugging, it visualizes bytecode parts with colors and converts assembler code to bytecode. Potential to evolve into a step-by-step debugger.

@ github

Grandpa Simulator

Simulates and debugs Grandpa implementations. In Finality Mode, create blockchains to see how Grandpa handles forks. In Comms Mode, inspect communication and state between nodes, and simulate events

@ github

RPC Docs & Client

Document and interact with JAM’s RPC structure. Connect to a running node, invoke RPCs, and see results in your browser with detailed documentation and an interactive client interface.
